"Summary: Latitude vs. X"

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis I first pulled data from OpenWeatherMap API to assemble a dataset on over 500 cities. Afther assembling the dataset, I used MAtplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: Temperture, Cloudiness, Wind Speed, and Humidity. This site provides the source data and visualiztions created as part of the analysis, as well as explanations and descriptions of any treads and correlations witnessed.
